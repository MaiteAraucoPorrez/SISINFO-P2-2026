# Proyecto Web - Sistemas de información

## Descripción
Proyecto web básico que muestra un mensaje "Hola Mundo" utilizando HTML5 y CSS3.

## Estructura del Proyecto
```
.
├── index.html      # Página principal
├── styles.css      # Estilos CSS
└── README.md       # Este archivo
```

## Archivos

### index.html
Contiene la estructura HTML básica con:
- Configuración de charset UTF-8
- Viewport responsive para dispositivos móviles
- Enlace a la hoja de estilos CSS
- Encabezado H1 con "Hola Mundo"

### styles.css
Archivo de estilos CSS (listo para personalización)

## Cómo usar

1. **Visualizar localmente:**
   - Abrir el archivo `index.html` directamente en su navegador web
   - O usar un servidor local como Live Server en VS Code

2. **Personalizar:**
   - Edita `index.html` para modificar el contenido
   - Agrega estilos en `styles.css` para personalizar la apariencia

## Tecnologías Utilizadas
- HTML5
- CSS3
